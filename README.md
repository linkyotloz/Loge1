## Loge

Loge is a "**L**ocal **O**ffline **G**ame **E**mulator" that supports a variety of game file types.
You are also able to run html, java, javascript, and c++ game files if you wish to do so.
For your information, files are processed localy and are not transfered to an external server. This also meaning that the emulator is able to run offline, **IF** you have installed it as an offline application.
